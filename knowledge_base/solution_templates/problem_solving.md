# Problem Solving Templates

## General Problem Solving Framework

### Step 1: Understand the Problem
- Read carefully and identify what is asked
- Note given information and unknowns
- Identify constraints and conditions
- Classify the problem type (algebra, calculus, probability, etc.)

### Step 2: Plan the Solution
- Recall relevant formulas and theorems
- Choose appropriate methods
- Consider special cases
- Look for patterns or similar problems

### Step 3: Execute the Solution
- Work step by step
- Show all calculations
- Keep track of units and signs
- Simplify at each step

### Step 4: Verify the Answer
- Check if answer makes sense
- Substitute back to verify
- Consider edge cases
- Check dimensions/units

## Quadratic Equation Template

**Given**: ax² + bx + c = 0

**Solution Steps**:
1. Identify coefficients: a = __, b = __, c = __
2. Calculate discriminant: D = b² - 4ac = __
3. Determine nature of roots:
   - D > 0: Two distinct real roots
   - D = 0: One repeated root
   - D < 0: Complex roots
4. Apply quadratic formula: x = (-b ± √D) / 2a
5. Simplify roots
6. Verify: Substitute roots back

## Limit Evaluation Template

**Given**: lim(x→a) f(x)

**Solution Steps**:
1. Try direct substitution: f(a) = ?
2. If defined → answer found
3. If indeterminate (0/0, ∞/∞):
   - Identify form type
   - Apply appropriate technique:
     - Factoring and cancellation
     - Rationalization
     - L'Hôpital's Rule
     - Standard limits
4. Evaluate simplified limit
5. State final answer

## Derivative Template

**Given**: Find dy/dx for y = f(x)

**Solution Steps**:
1. Identify function type:
   - Basic function → use table
   - Composite → use chain rule
   - Product → use product rule
   - Quotient → use quotient rule
   - Implicit → implicit differentiation
2. Apply appropriate rule(s)
3. Simplify the expression
4. Verify by checking dimensions

## Probability Template

**Given**: Probability problem with events

**Solution Steps**:
1. Define sample space S
2. Identify events A, B, etc.
3. Count favorable outcomes or use formulas:
   - P(A∪B) = P(A) + P(B) - P(A∩B)
   - P(A∩B) = P(A)·P(B|A)
   - For independent: P(A∩B) = P(A)·P(B)
4. Calculate probability
5. Verify: 0 ≤ P ≤ 1

## Optimization Template

**Given**: Maximize/minimize a quantity

**Solution Steps**:
1. Define variables
2. Write objective function f(x)
3. Identify constraints
4. Express in one variable
5. Find domain
6. Differentiate: f'(x) = 0
7. Find critical points
8. Test for max/min:
   - Second derivative test
   - Or first derivative test
9. Check endpoints
10. State optimal value

## Matrix Problem Template

**Given**: Matrix operations or equation

**Solution Steps**:
1. Identify matrix dimensions
2. Check operation compatibility
3. Apply rules:
   - Addition: element-wise
   - Multiplication: row × column
   - Inverse: adj(A)/det(A)
4. Perform calculations
5. Verify: Check with properties
