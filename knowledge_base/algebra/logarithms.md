# Logarithms and Exponentials

## Definition
If aˣ = N, then x = logₐN (a > 0, a ≠ 1, N > 0)

## Laws of Logarithms
1. **Product Rule**: logₐ(MN) = logₐM + logₐN
2. **Quotient Rule**: logₐ(M/N) = logₐM - logₐN
3. **Power Rule**: logₐ(Mⁿ) = n·logₐM
4. **Change of Base**: logₐN = logᵦN / logᵦa = ln N / ln a

## Important Properties
- logₐ1 = 0
- logₐa = 1
- logₐ(aˣ) = x
- a^(logₐx) = x
- logₐb × logᵦa = 1
- logₐb = 1/logᵦa

## Common Logarithms
- **Natural Logarithm**: ln x = logₑx (base e ≈ 2.71828)
- **Common Logarithm**: log x = log₁₀x (base 10)

## Exponential Functions
### Properties of Exponentials (a > 0, a ≠ 1)
- aˣ · aʸ = aˣ⁺ʸ
- aˣ / aʸ = aˣ⁻ʸ
- (aˣ)ʸ = aˣʸ
- aˣ · bˣ = (ab)ˣ
- a⁰ = 1
- a⁻ˣ = 1/aˣ

### Special Values (base e)
- e⁰ = 1
- ln 1 = 0
- ln e = 1
- e^(ln x) = x
- ln(eˣ) = x

## Exponential Growth and Decay
- Growth: N(t) = N₀ · eᵏᵗ (k > 0)
- Decay: N(t) = N₀ · e⁻ᵏᵗ (k > 0)
- Half-life: t₁/₂ = ln 2 / k

## Solving Logarithmic Equations
### Method 1: Using Definition
logₐx = b ⟹ x = aᵇ

### Method 2: Combining Logs
logₐx + logₐy = logₐz ⟹ xy = z

### Method 3: Substitution
Let logₐx = y, solve for y, then find x

## Solving Exponential Equations
1. Take log of both sides
2. Use log rules to simplify
3. Solve for variable

**Example**: 2ˣ = 8 → x·ln 2 = ln 8 → x = 3

## Domain Restrictions
- For logₐx: x > 0
- For aˣ: all real x (a > 0, a ≠ 1)
