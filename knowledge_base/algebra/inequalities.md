# Inequalities

## Types of Inequalities
1. **Linear Inequalities**: ax + b > 0, ax + b ≤ 0
2. **Quadratic Inequalities**: ax² + bx + c > 0
3. **Rational Inequalities**: P(x)/Q(x) > 0
4. **Absolute Value Inequalities**: |x - a| < b

## Properties of Inequalities
- Adding/subtracting same value preserves inequality
- Multiplying/dividing by positive number preserves inequality
- Multiplying/dividing by negative number REVERSES inequality
- Squaring preserves inequality only if both sides are non-negative

## Solving Quadratic Inequalities
For ax² + bx + c > 0 (or < 0):
1. Find roots using quadratic formula
2. Determine sign of 'a' (parabola opens up if a > 0, down if a < 0)
3. Draw number line and mark roots
4. Test intervals
5. Write solution set

**Key Insight**: 
- If a > 0: ax² + bx + c > 0 outside the roots, < 0 between roots
- If a < 0: ax² + bx + c > 0 between the roots, < 0 outside roots

## Absolute Value Inequalities
- |x| < a ⟹ -a < x < a (if a > 0)
- |x| > a ⟹ x < -a OR x > a (if a > 0)
- |x - c| < a ⟹ c - a < x < c + a
- |x - c| > a ⟹ x < c - a OR x > c + a

## Rational Inequalities
For (x-a)(x-b)/((x-c)(x-d)) > 0:
1. Find all zeros and undefined points
2. Create sign chart
3. Test each interval
4. Combine intervals based on sign requirement
5. Exclude points where denominator = 0

## AM-GM-HM Inequality
For positive numbers a₁, a₂, ..., aₙ:
$$\frac{a_1 + a_2 + ... + a_n}{n} \geq \sqrt[n]{a_1 \cdot a_2 \cdot ... \cdot a_n} \geq \frac{n}{\frac{1}{a_1} + \frac{1}{a_2} + ... + \frac{1}{a_n}}$$

**AM ≥ GM ≥ HM** (Arithmetic Mean ≥ Geometric Mean ≥ Harmonic Mean)

Equality holds when all numbers are equal.

## Cauchy-Schwarz Inequality
$$(a_1b_1 + a_2b_2 + ... + a_nb_n)^2 \leq (a_1^2 + a_2^2 + ... + a_n^2)(b_1^2 + b_2^2 + ... + b_n^2)$$

## Triangle Inequality
For any real numbers a and b:
$$|a + b| \leq |a| + |b|$$
$$|a - b| \geq ||a| - |b||$$
